# Font game (FOT-UDMarugo_SmallPr6-E SDF) supports: á â ê ô ú à è ù ò etc. (Latin-1 U+00C0-U+00FF)
# Font game does NOT support: ơ ư ắ ậ ẻ đ etc. (Latin Extended Additional U+1E00-U+1EFF)
# These substitutions replace ONLY unsupported chars with the closest supported equivalent.
# Result: "Nhận" -> "Nhân", "chơi" -> "choi" — readable, no boxes.
